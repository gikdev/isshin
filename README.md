# 一心 (Isshin)

Your focus companion.

## 📋 Prerequisites

- [Taskfile](https://taskfile.dev/)
- [Vite+](https://viteplus.dev/)

## ⚡ Quick Start

```bash
task setup
```

## 🧭 Development Workflow

### Available Commands

All commands are discoverable by running:

```bash
task --list-all
```

### `.dev/` Folder

The `.dev/` folder holds files that the developer(s) use/need. Which include:
- `Isshin.smmx`: The mindmap file, opened with **SimpleMind Pro**
- `Isshin.sketch`: The design file, opened with **Icons8 Lunacy**
